# Onalenna Rehabilitation Hub

Botswana's first comprehensive rehabilitation center dedicated to transforming lives through evidence-based addiction, alcoholism, and mental health treatment.

## About

The Onalenna Rehabilitation Hub website serves as the digital gateway to Botswana's pioneering rehabilitation center. Our mission is to provide hope, healing, and transformation for individuals and families affected by addiction and mental health challenges.

### Key Features

- **Comprehensive Information**: Learn about our treatment programs, services, and approach
- **Crisis Support**: Immediate access to crisis hotlines and emergency resources
- **Donation Platform**: Support our mission through secure online donations
- **Family Resources**: Support systems for families affected by addiction
- **Contact & Consultation**: Easy access to schedule consultations and get help

## Services

- **Addiction Treatment**: Evidence-based rehabilitation programs
- **Mental Health Support**: Professional counseling and therapy services
- **Family Counseling**: Support for families affected by addiction
- **Crisis Intervention**: 24/7 crisis support and emergency services
- **Community Outreach**: Education and prevention programs

## Technology Stack

- **Frontend**: React 18 with TypeScript
- **UI Framework**: shadcn/ui components with Tailwind CSS
- **Backend**: Express.js with Node.js
- **Database**: PostgreSQL with Drizzle ORM
- **Build Tool**: Vite
- **Hosting**: Replit

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open your browser to `http://localhost:5000`
